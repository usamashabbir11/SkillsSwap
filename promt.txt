WHAT TO BUILD:

1. Create backend/controllers/suggestionController.js:
   - ONE function: getSuggestions(req, res)
   - Fetches all users except the logged-in user
   - For each user calculates a COMBINED score using:
     * AI match score (cosine similarity) — weight 50%
     * Distance score (closer = higher score, max 100km range) — weight 30%
     * Rating score (avgRating / 5 * 100) — weight 20%
   - Returns top 5 users sorted by combined score descending
   - Each user in response includes: name, email, city, profileImage, skillsOffered, skillsRequired, avgRating, courses count, combinedScore, distanceKm
   - If user has no location, distance weight is redistributed to match score

2. Create backend/routes/suggestionRoutes.js:
   - GET /suggestions — protected with getLoggedInUser middleware

3. Register in backend/app.js:
